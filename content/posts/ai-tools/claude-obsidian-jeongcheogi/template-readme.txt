# 학습용 Obsidian 볼트 — base 양식

이 폴더는 학습용 Obsidian 볼트의 출발점이다. 도구·운영 차원의 보편 규칙(Edit 가드레일·메타블록 양식 등)만 미리 들어가 있고, 도메인 특화 항목(학습 주제·폴더 구조·모드·H2 네이밍·정제 원칙 등)은 사용자가 AI와 대화하며 채워나간다.

## 사용법

이 폴더에서 Claude Code를 실행하고 다음과 같이 요청한다.

> "이 README의 체크리스트를 따라 내 학습 볼트를 빌드해줘"

Codex CLI 사용자는 먼저 `mv CLAUDE.md AGENTS.md`로 파일명만 변경한다.

## 채울 항목

AI가 차례로 묻고 답을 받아 `CLAUDE.md`·규칙 파일·폴더·메타블록 양식을 사용자 도메인에 맞게 작성한다.

- [ ] **학습 주제·도메인** — 시험 / 강의 / 자격증 / 자기학습 주제 등
- [ ] **자료 형태** — 기출 PDF / 교재 / 강의 노트 / 자기 정리 등 (있는 경우)
- [ ] **학습 단위** — 회차 / 챕터 / 단원 / 일자 등
- [ ] **모드 결정** — 어떤 사이클을 운영할지 (복기 / 오답반복 / 랜덤퀴즈 등)
- [ ] **폴더 구조** — 위 결정에 따라 만들 폴더 (개념노트·복기·자료·규칙 등)
- [ ] **메타블록 필드** — 기본(출제·오답·헷갈림·최근·이력)을 그대로 쓸지·바꿀지·추가할지
- [ ] **H2 네이밍 규칙** — 괄호·부가설명 허용 여부
- [ ] **위키링크 연결 레벨** — 어느 헤딩까지 잇는지 (기본 H2 / 조건부 H3 등)
- [ ] **정제 원칙** — 보조 섹션·이모지·코드 분량·중복 통합 등 본인 취향
- [ ] **모델 분리 정책** — Opus 메인 / Sonnet 가벼운 작업 등
- [ ] **응답 스타일** — 한국어 평어체 / 영어 / 격식 톤 등
